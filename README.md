## How to setup

최신 버전의 node.js 와 npm를 설치하길 권장합니다

```sh
git clone https://github.com/smilu97/oop_term_react
npm install
```

## How to build

```sh
npm run build
```

## How to run to debug

```sh
npm start
```

http://localhost:3000 에 실행됩니다.
